# VIRALPULSE MASTER CONTEXT
**Version 1.0** — May 6, 2026  
**Owner**: Jay Caesar

### 1. Core Objective (The North Star)

**Primary Goal**:  
**Maximize real follower growth** (measurable net increase in followers) on X.

This is the **#1 KPI** everything in the app optimizes for.  
Content quality, virality, engagement, and aesthetics are secondary and must serve follower growth.

**Default Philosophy (Standard Setting)**:  
**Pure Growth Mode** = Aggressive growth by default.  
The app should “do whatever it takes” (within X rules) to grow the account — including changing niches, posting breaking news, using viral formats, trend riding, etc.

Niche selection is **optional**. The user can choose Niche-Loyal Mode if they want to stay restricted.

### 2. User Control

Users have full authority:
- Switch between Pure Growth Mode and Niche-Loyal Mode
- Set aggressiveness level (1–10)
- Set weekly follower target
- Define niches / exclusions / tone examples
- Approve or pause any post

### 3. Architecture (Brain + Galaxies)

- **Brain / Orchestrator** → Top-level central intelligence that manages everything
- **Galaxies** → Independent strategy worlds that can run in parallel and evolve
- Clean separation: Brain manages Galaxies. Galaxies focus on strategy.

### 4. Current Tech Stack
- Next.js 15 (App Router) + TypeScript + Tailwind
- Brain: Claude Sonnet 4.6
- X: Official X API v2
- Future: Supabase, Image Generation, multi-platform support

### 5. Important Rules
- Always stay fully compliant with X’s TOS
- Prioritize original content and images when possible
- Human approval workflow is required (user can adjust strictness)
- Growth must be real and sustainable
